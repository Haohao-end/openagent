🚀 如何使用

  • 进入工作流详情页面：/space/workflows/[workflow_id]
  • 添加或点击 Python 代码节点
  • 右侧面板会显示专业的代码编辑器
  • 开始编写代码，实时查看语法高亮和错误提示
  • 代码会自动保存，无需手动操作

⚠️ 验证规则

  • 必须满足：定义 def main(params): 函数
  • 必须满足：正确的缩进（4 个空格）
  • 必须满足：括号必须匹配
  • 建议遵守：使用 params.get('key', default) 代替直接访问
  • 建议遵守：添加类型转换和异常处理
